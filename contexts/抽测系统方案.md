# 象山县信息科技统考系统 - 产品需求规格说明书

> **版本**: v1.1  
> **日期**: 2026-01-16  
> **状态**: 待确认

---

## 一、项目背景

每年象山县教育局需组织全县中小学校进行一次统一的信息科技学科考试，目的是：

1. 对各学校的教学质量进行评估
2. 形成全县学校排名
3. 为教育决策提供数据支撑

---

## 二、系统角色

| 角色         | 权限说明                                                             |
| ------------ | -------------------------------------------------------------------- |
| **教研员**   | 创建县考、组卷、指派班级、设置评卷人员、审核成绩、查看分析、历史存档 |
| **评卷教师** | 批阅分配的操作题（匿名批阅，看不到学生信息）                         |
| **参考学生** | 参加县考答题（打字/理论/操作），**无法查看县考成绩**                 |
| **普通教师** | **无县考权限**，看不到任何县考信息                                   |

---

## 三、功能模块

### 3.1 教研员首页

新增专属菜单，包含：

- **区域考试管理**：新增/编辑/删除县考
- **评卷管理**：分配评卷任务、查看进度
- **成绩分析**：全县成绩查询与可视化
- **历史存档**：历年县考数据查询

### 3.2 创建区域考试

**基本信息配置：**

| 配置项   | 说明                                               |
| -------- | -------------------------------------------------- |
| 考试名称 | 如"2026 年象山县小学六年级信息科技统考"            |
| 学校类型 | 小学(1-6 年级) / 初中(7-9 年级) / 高中(10-12 年级) |
| 考试年级 | 根据学校类型显示对应年级选项                       |

**组卷功能：**

- 从现有题库选择题目（选择题、判断题、打字题、操作题）
- 设置各题分值
- 支持固定顺序、随机排序、随机抽题三种模式

### 3.3 指派学校班级

**选择规则：**

- 系统自动显示对应类型的所有学校（如选择"初中"则显示所有初中）
- 显示该年级的所有班级
- **每所学校只能选择一个班级**参加考试

**界面示意：**

```
学校         班级（点击选择）
─────────────────────────────────────
滨海学校     [1班] [2班✓] [3班] [4班]
测试学校     [1班✓]
大目湾学校   [1班] [2班] [3班✓] [4班] [5班] [6班]
丹城二中     [1班✓] [2班] [3班] [4班]
...
```

### 3.4 考试状态管理

| 状态       | 说明                          |
| ---------- | ----------------------------- |
| **草稿**   | 初始状态，可编辑              |
| **已开启** | 学生可以答题                  |
| **已关闭** | 学生无法再提交/修改，等待评卷 |
| **已发布** | 成绩已审核发布                |

### 3.5 学生答题流程

**登录逻辑：**

1. 学生登录系统
2. 系统判断该学生是否在县考班级
3. 如果在县考班级且考试已开启且未完成：**自动进入县考答题页**
4. 完成后或考试关闭后：恢复正常学校课程

**答题规则：**

| 规则          | 说明                           |
| ------------- | ------------------------------ |
| 断点续考      | 中途退出，下次登录自动恢复     |
| 操作题可重传  | 可删除重新上传，与现有功能一致 |
| 未提交按 0 分 | 操作题未上传默认 0 分          |
| 考试关闭      | 关闭后学生无法再提交/修改      |

### 3.6 匿名评卷系统

**评卷人员配置：**

- 从现有教师账号中选择评卷老师
- 点击"平均分配"自动均分任务
- 示例：1000 份 ÷ 5 人 = 每人 200 份

**评卷规则：**

- **随机分配**：学生答卷随机分配，不回避本校
- **匿名批阅**：评卷界面不显示学生姓名、学校
- **批阅方式**：查看 PDF + 按评分标准打分

**评卷入口：**

评卷教师登录后，在"选择校区"弹窗中多出"**县考评卷**"选项，点击进入评卷工作台。

### 3.7 成绩审核与发布

**流程：**

1. 所有操作题评卷完成
2. 教研员进入审核页面
3. 查看各学校成绩汇总、异常分数
4. 确认无误后点击"发布成绩"

### 3.8 全县成绩分析

**分析维度：**

| 分析项   | 说明                   |
| -------- | ---------------------- |
| 学校排名 | 按平均分排名的柱状图   |
| 分数分布 | 全县学生分数分布直方图 |
| 题目分析 | 各题正确率对比         |
| 导出功能 | 支持 Excel 导出        |

**排名数据表示例：**

| 排名 | 学校名称 | 参考人数 | 平均分 | 最高分 | 最低分 | 及格率 |
| ---- | -------- | -------- | ------ | ------ | ------ | ------ |
| 1    | 滨海学校 | 45       | 92.5   | 100    | 78     | 100%   |
| 2    | 丹城二中 | 52       | 89.3   | 98     | 65     | 96%    |

### 3.9 历史存档

- 查询历年县考记录
- 查看历史成绩和分析
- 支持不同年份数据对比

---

## 四、数据隔离

县考数据与学校日常课程数据**完全隔离存储**，使用独立的数据表：

- `biz_county_exam` - 县考主表
- `biz_county_exam_question` - 县考题目
- `biz_county_exam_class` - 县考班级
- `biz_county_exam_answer` - 县考答题记录
- `biz_county_exam_student` - 学生状态
- `biz_county_exam_grader` - 评卷分配
